# David Zaev — Portfolio

My personal portfolio site. Built from scratch with HTML, CSS, and vanilla JavaScript.

**[→ Live site](https://david-zaev.github.io/portfolio/)**

## Features

- Clean dark theme with responsive design
- Interactive terminal (try typing commands!)
- Mobile-friendly navigation
- Click-to-copy email contact

## Tech

- HTML5 / CSS3 / JavaScript
- No frameworks or dependencies
- Hosted on GitHub Pages

## Run locally

Open `index.html` in a browser, or:

```bash
npx serve .
```
